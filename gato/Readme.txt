Crea un directorio llamado greeter en el directorio src.

$ mkdir greeter

posiciónese en el directorio recién creado

$cd greeter

coloque la carpeta "gato" dentro de greeter
el comando cp puede ayudar a colocar archivos de un destino a otro

$ cp [gato] [ubicacion/deLacarpetaGreeter]

una vez copiado, posicionese dentro de la carpeta gato

$ cd /ubicacion/gato

liste los archivos dentro de la carpeta gato

$ ls

$ gato.go log.txt Readme.txt tablero.txt

ejecute gato.go con el siguiente comando

go run gato.go
